# 🧮 Calculator Project

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear button
- Delete button
- Responsive design
- Clean user interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
